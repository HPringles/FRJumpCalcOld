INSTALLATION INSTRUCTIONS

RUN FRJumpCalcSetup.exe

THIS WILL NOT CREATE A SHORTCUT

TO CREATE A SHORTCUT:

Open windows explorer(file manager) and type %LOCALAPPDATA%  (with the %'s) into
the navigation bar - In the search bar- search for frjump 

Open the folder labeled frjump

RIGHT CLICK on the file frjump.exe and select send to > desktop(create shortcut)

Check the shortcut is on the desktop
